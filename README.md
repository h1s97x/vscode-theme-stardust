# Stardust Theme

> A dark VS Code theme inspired by the cosmos ✨

![Version](https://img.shields.io/badge/version-0.1.0-blue)
![License](https://img.shields.io/badge/license-MIT-green)
![VS Code](https://img.shields.io/badge/VS%20Code-1.74%2B-blue)

## 🌟 Features

- **Deep dark background** inspired by the night sky
- **Carefully selected colors** for long coding sessions
- **Optimized readability** with reduced eye strain
- **Multi-language support** with syntax highlighting

## 📦 Installation

### From VS Code Marketplace (Coming Soon)

1. Open VS Code
2. Go to Extensions (`Ctrl+Shift+X` / `Cmd+Shift+X`)
3. Search for **"Stardust Theme"**
4. Click **Install**
5. Select the theme: `Preferences > Color Theme > Stardust`

### From Source

```bash
# Clone the repository
git clone https://github.com/h1s97x/stardust-theme.git

# Copy to VS Code extensions folder
cp -r stardust-theme ~/.vscode/extensions/

# Restart VS Code and select the theme
```
